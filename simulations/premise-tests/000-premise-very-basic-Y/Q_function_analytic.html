Analytic Q-Function for MVN with Missing Data

1 Setup

For multivariate normal data with missing values, partition the data into observed and missing components.

Conditional distribution of missing given observed:

\mu_{m|o} = \mu_m + \Sigma_{mo} \Sigma_o^{-1} (Y_o - \mu_o) \tag{1}

\Sigma_{m|o} = \Sigma_m - \Sigma_{mo} \Sigma_o^{-1} \Sigma_{mo}^T \tag{2}

2 Q-Function Definition

The Q-function is the expected complete-data log-likelihood, where the expectation is over the conditional distribution of missing data:

Q(\hat{\theta} | \hat{\theta}) = E_{Y^m | Y^o, \hat{\theta}} \left[\log p(Y | \hat{\theta})\right] \tag{3}

3 Derivation

3.1 Step 1: Factor the Joint Density

Q(\hat{\theta} | \hat{\theta}) = E_{Y^m | Y^o, \hat{\theta}} \left[\log p(Y^m | Y^o, \hat{\theta}) + \log p(Y^o | \hat{\theta})\right] \tag{4}

Since \log p(Y^o | \hat{\theta}) does not depend on Y^m, it passes through the expectation:

= E_{Y^m | Y^o, \hat{\theta}} \left[\log p(Y^m | Y^o, \hat{\theta})\right] + \log p(Y^o | \hat{\theta}) \tag{5}

3.2 Step 2: Conditional Log-Likelihood

The conditional distribution Y^m | Y^o, \hat{\theta} is multivariate normal with mean \mu_{m|o} and covariance \Sigma_{m|o}. Therefore:

\log p(Y^m | Y^o, \hat{\theta}) = -\frac{q}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o}| - \frac{1}{2}(Y^m - \mu_{m|o})^T \Sigma_{m|o}^{-1}(Y^m - \mu_{m|o}) \tag{6}

where q is the dimension of the missing data.

3.3 Step 3: Take Expectation

Taking the expectation over Y^m | Y^o, \hat{\theta}:

E_{Y^m | Y^o, \hat{\theta}}\left[\log p(Y^m | Y^o, \hat{\theta})\right] = -\frac{q}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o}| - \frac{1}{2}E_{Y^m | Y^o, \hat{\theta}}\left[(Y^m - \mu_{m|o})^T \Sigma_{m|o}^{-1}(Y^m - \mu_{m|o})\right] \tag{7}

3.4 Step 4: Evaluate the Quadratic Form

Using the identity for quadratic forms of centered Gaussian vectors:

E[X^T A X] = \text{tr}(A \cdot \text{Var}(X)) \quad \text{when } E[X] = 0 \tag{8}

Since Y^m | Y^o, \hat{\theta} \sim N(\mu_{m|o}, \Sigma_{m|o}), we have Y^m - \mu_{m|o} centered with variance \Sigma_{m|o}:

E\left[(Y^m - \mu_{m|o})^T \Sigma_{m|o}^{-1}(Y^m - \mu_{m|o})\right] = \text{tr}\left(\Sigma_{m|o}^{-1} \Sigma_{m|o}\right) = \text{tr}(I_q) = q \tag{9}

3.5 Step 5: Combine Terms

Substituting back:

E_{Y^m | Y^o, \hat{\theta}}\left[\log p(Y^m | Y^o, \hat{\theta})\right] = -\frac{q}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o}| - \frac{q}{2} \tag{10}

4 Result (Single Observation)

The Q-function contribution from the missing data for a single observation i is:

Q_i(\hat{\theta} | \hat{\theta})_{\text{mis}} = -\frac{q_i}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o,i}| - \frac{q_i}{2} \tag{11}

Or more compactly:

Q_i(\hat{\theta} | \hat{\theta})_{\text{mis}} = -\frac{q_i}{2}\left(1 + \log(2\pi)\right) - \frac{1}{2}\log|\Sigma_{m|o,i}| \tag{12}

where:

  • q_i = number of missing variables for observation i
  • \Sigma_{m|o,i} = conditional covariance matrix for observation i’s missing pattern

5 Aggregation Across Observations

For monotone missingness, observations fall into groups with the same missing pattern. Let:

  • K = number of distinct missing patterns
  • n_k = number of observations with pattern k
  • q_k = number of missing variables in pattern k
  • \Sigma_{m|o,k} = conditional covariance for pattern k (same for all obs in group k)

The total Q-function contribution from missing data is:

Q(\hat{\theta} | \hat{\theta})_{\text{mis}} = \sum_{k=1}^{K} n_k \left[ -\frac{q_k}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o,k}| - \frac{q_k}{2} \right] \tag{13}

This simplifies to:

Q(\hat{\theta} | \hat{\theta})_{\text{mis}} = -\frac{1}{2}\sum_{k=1}^{K} n_k \left[ q_k(1 + \log(2\pi)) + \log|\Sigma_{m|o,k}| \right] \tag{14}

6 Interpretation

The Q-function contribution from missing data depends on:

  1. q_k: The dimension of missing data in each pattern
    • Comes from \text{tr}(\Sigma_{m|o}^{-1}\Sigma_{m|o}) = q_k
  2. \log|\Sigma_{m|o,k}|: The log-determinant of conditional covariance for each pattern
    • Captures uncertainty in the conditional distribution
  3. n_k: The count of observations in each missing pattern
    • Weights each pattern’s contribution

7 Connection to Entropy

This result is the negative differential entropy of the conditional distribution:

H(Y^m | Y^o) = \frac{q}{2}\left(1 + \log(2\pi)\right) + \frac{1}{2}\log|\Sigma_{m|o}| \tag{15}

So:

Q_i(\hat{\theta} | \hat{\theta})_{\text{mis}} = -H(Y_i^m | Y_i^o) \tag{16}

8 Numerical Verification via Gauss-Hermite Quadrature

We verify the analytic formula using Gauss-Hermite quadrature, which provides machine-precision integration for Gaussian expectations.

Simplified test case:

  • p = 4 dimensional MVN
  • Single missing pattern: observe (Y_1, Y_2), miss (Y_3, Y_4)
  • q = 2 missing dimensions

library(MASS)

8.2 Partition into Observed/Missing

Code 
# Observe first 2, miss last 2
obs_idx <- 1:2
mis_idx <- 3:4
q <- length(mis_idx)

# Fixed observed values
Y_obs <- c(0.5, 1.5)

# Partition parameters
mu_o <- mu[obs_idx]
mu_m <- mu[mis_idx]
Sigma_oo <- Sigma[obs_idx, obs_idx]
Sigma_mm <- Sigma[mis_idx, mis_idx]
Sigma_mo <- Sigma[mis_idx, obs_idx]
Sigma_om <- Sigma[obs_idx, mis_idx]

# Conditional distribution parameters (Eqs. 1-2)
Sigma_oo_inv <- solve(Sigma_oo)
mu_m_given_o <- mu_m + Sigma_mo %*% Sigma_oo_inv %*% (Y_obs - mu_o)
Sigma_m_given_o <- Sigma_mm - Sigma_mo %*% Sigma_oo_inv %*% Sigma_om

8.4 Gauss-Hermite Quadrature

For a Gaussian integral with Y^m \sim N(\mu_{m|o}, \Sigma_{m|o}), we use the transformation via Cholesky decomposition: Y^m = \mu_{m|o} + \sqrt{2} L^T z where L^T L = \Sigma_{m|o}.

Code 
# Gauss-Hermite quadrature points and weights
n_quad <- 20
gh <- gauss.quad(n_quad, kind = "hermite")

# Cholesky decomposition
L <- chol(Sigma_m_given_o)

# Log conditional density function
log_cond_density <- function(Y_m) {
  diff <- Y_m - as.vector(mu_m_given_o)
  Sigma_inv <- solve(Sigma_m_given_o)
  -q/2 * log(2*pi) - 0.5 * log(det(Sigma_m_given_o)) -
    0.5 * sum(diff * (Sigma_inv %*% diff))
}

# Tensor product grid for 2D integration
grid <- expand.grid(z1 = gh$nodes, z2 = gh$nodes)

Q_numerical <- 0
for (i in 1:nrow(grid)) {
  z <- c(grid$z1[i], grid$z2[i])

  # Transform: Y_m = mu_m|o + sqrt(2) * L' * z
  Y_m <- as.vector(mu_m_given_o) + sqrt(2) * t(L) %*% z

  # Weight = w1 * w2 / pi
  w <- gh$weights[which(gh$nodes == grid$z1[i])] *
       gh$weights[which(gh$nodes == grid$z2[i])] / pi

  Q_numerical <- Q_numerical + w * log_cond_density(Y_m)
}

cat("Gauss-Hermite Q_mis =", Q_numerical, "\n")
Gauss-Hermite Q_mis = -2.39159

9 Summary

The Gauss-Hermite verification confirms the analytic formula to machine precision. The key result (Eq. 11) is:

Q_i(\hat{\theta} | \hat{\theta})_{\text{mis}} = -\frac{q}{2}\log(2\pi) - \frac{1}{2}\log|\Sigma_{m|o}| - \frac{q}{2}

This can be computed directly from the conditional covariance \Sigma_{m|o} without any Monte Carlo sampling.
